## ❤️ Heart Disease Prediction System

## 📌 Project Overview
Heart Disease Prediction System is a Machine Learning-based web application that predicts whether a person is at high risk or low risk of heart disease based on various medical parameters. The application uses a trained K-Nearest Neighbors (KNN) model and provides real-time predictions through an interactive Streamlit web interface.

The goal of this project is to assist in early detection of heart-related risks using patient health information.

---

## 🚀 Features :
-  User-friendly Streamlit Web Interface,
- Real-time Heart Disease Prediction,
- Data Preprocessing and Feature Engineering,
- Machine Learning Model Training using KNN,
- Feature Scaling using StandardScaler,
- One-Hot Encoding for Categorical Features,
- Risk Classification (High Risk / Low Risk),
- Interactive Input Controls (Sliders, Dropdowns, Numeric Fields)
